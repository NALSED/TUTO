### `Variables`

#### 1️⃣ Snake case : my_variable_name
#### 2️⃣ Ne pas utiliser le mots clés de python (else, if, for, while, return, True, False) 
#### 3️⃣ [NOM VARIABLE] [OPERATEUR D'AFFECTATION] [VALEUR]
#### Portée Global🔴
#### Portée Local🟢


            device_name = "router 1" #🔴

            def  test():
                test_value = "test OK" #🟢
                print(test_value)
                print(device_name)

            test()

#### Slicing

      >>> nom = "Jean"
      >>> nom [0]
      'J'
      >>> nom [0:3]
      'Jea'
      >>> nom [0:1]
      'J'
      >>> nom [0:2]
      'Je'
      >>> nom [0:4]
      'Jean'
